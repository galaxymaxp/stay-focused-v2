SESSION AIM

The session aim is to compare observations, records, and conclusions without adding facts.
Short note: keep each record tied to the evidence that produced it.

- observation means a noticed detail from the source activity
- record means the written trace kept for later checking
- conclusion means a statement drawn from the available record

  working terms

1. Input item - material or detail received before work begins
2. Check point - place where the item is compared with a rule
3. Output note - result written after the check point
4. Review mark - sign that the output note was inspected

Process sketch

1. First read the item label.
2. Then compare the status with the expected condition.
3. Next count the units that remain.
4. Finally write one output note.

OCR noise: rn may look like m; 0 may look like O; l may look like 1.
